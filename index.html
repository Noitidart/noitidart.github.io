About Me

I have been programming for many years. I started dabbling with HTML in 1999 and by 2000 I was well into JavaScript. I kept up with it as a hobby making scripts that looked like flash, I was a big fan of bratta.com which later became DHTMLCentral.com, these sites are long gone now. They made very Flash like stuff with HTML, JavaScript, and CSS, I remember we called this combination DHTML back then. I continued to make snippets and websites for fun but JavaScript didn't have much of a future back then.


Then in 2004 I got heavily into VB6. I made applications to interact with various websites. I then got into maintaining a forum based on IP.Board software, it involved PHP, MySQL, and yes... JavaScript!! All that time I had spent with now looked so awesome. People would tell me Flash was the future and look now, JavaScript was taking off. I made many hooks and modules for the IPB system and this is where I really improved my coding. As what I was making was being implemented and used by thousands of users.


Early on I was using Mozilla Firebird, which was the predecessor of Firefox. I later moved to Firefox in early 2005. I was still doing VB6 applications but then came across Firefox extensions and AMO. I was now able to write advanced software with JavaScript! I have been working with Firefox addons ever since.


I was using builder.addons.mozilla.org for the longest time because I could sit down anywhere and program and test with a click. However they decided to shut that down and then I looked for version control and came across GitHub, it has been an awesome experience. Currently you can find me helping people with Firefox addon development at StackOverflow, MozillaZine, Mozilla Addons Forum, and Mozilla Firefox Extension Dev Mailing List.
