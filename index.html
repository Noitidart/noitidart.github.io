Gists

Contained here are useful codes to help you build Firefox addons. I chose Gists as a source to share because edits are version controlled, easy to fork to repo, and can discuss with fellow programmers in the section below.


All Firefox related Gists are prefixed with _ff-addon-. This is followed by one of the following subcategories:

Snippet

These are codes you can copy and paste to "Scratchpad" in Firefox. Set the "Environment" to "Browser" and run.

Template

These are full addons but very basic, ready to fork and personalize. I start all my addons based on one of these templates.
